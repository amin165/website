HTML Terms

HTML
Hyper Text Markup Language
HTML element
Start tag, some content, and an end tag
HTML tag
Command in brackets at start and end of text
angle bracket
HTML tags are enclosed in angle brackets
HTML attribute
Provide additional information about the HTML elements
HTML comment
Text that is not printed out, but rather for the readers information
HTML object element
An external resource that can be treated as an image
HTML special character
A character that cannot be printed out by itself because of its reserved functions
